##### Vue Starter

> vue.js 프로젝트를 시작할 수 있도록 돕는 헬퍼



##### 사용 프로그램

- NPM
- VSCode
- Webpack



##### VSCode 플러그인

- Vetur - Vue.js 플러그인
- ESLint - Javascript 문법 검사 플러그인
- TSLint - Typescript 문법 검사 플러그인
- Auto Close tag - HTML 태그 자동 닫기 플러그인
- Night Owl - 코드 하이라이팅 플러그인
- Live Server - 정적파일 웹서버 플러그인
- Material Icon Theme - 파일 확장자별 아이콘 표시
